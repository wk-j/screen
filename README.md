## Screen

[![NuGet](https://img.shields.io/nuget/v/wk.Screen.svg)](https://www.nuget.org/packages/wk.Screen)

Execute command in remote shell

## Installation

```bash
dotnet tool install -g wk.Screen
```

## Usage

Terminal 1

```bash
wk-screen --init
```

Terminal 2

```bash
wk-screen --command ls
```
